# git-branches
Review branches in git

Command line options:

```
  -p, --path           The path to the local git repository

  -m, --main           (Default: origin/master) The main branch of the git repository

  -b, --branch         Only include branches which contain this (e.g. -b 123456)

  -c, --contributor    Only include branches which contain this contributor (e.g. -c udv)

  -v, --verbosity      (Default: Normal) Level of details in the output (Compact, Normal, Contributors, Logs)

  --help               Display this help screen.

  --version            Display version information.
```
